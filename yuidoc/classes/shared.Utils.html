addChildren

(
  • categories
)

Defined in status/assets/js/shared/utils.js:12

Adds children to a categories primary key object.

For example,


{
    1: {
        pk: 1,
        name: 'Category 1',
        parent: null,
    },
    2: {
        pk: 2,
        name: 'Category 2',
        parent: 1
    }
}
becomes 

{
    1: {
        pk: 1,
        name: 'Category 1',
        parent: null,
        children: [{
            pk: 2,
            name: 'Category 2',
            parent: 1
        }]
    },
    2: {
        pk: 2,
        name: 'Category 2',
        parent: 1
    }
}

  • categories Object
